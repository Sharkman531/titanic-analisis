# Análisis del Titanic 🚢

Análisis exploratorio y modelo predictivo sobre los datos del Titanic
de la competencia de Kaggle.

## Contenido
- Exploración y visualización de datos
- Limpieza de valores faltantes
- Modelo Random Forest con 81% de precisión
- Predicciones subidas a Kaggle

## Tecnologías
Python | pandas | matplotlib | seaborn | scikit-learn

## Principales hallazgos
- Las mujeres sobrevivieron el 74% de las veces vs 19% los hombres
- Primera clase tuvo 63% de supervivencia vs 24% en tercera clase
- Variables más predictivas: Fare, Sex y Age
